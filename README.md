### **Cấu hình dev:**

- Nodejs (v14.1.0)
- npm (v7.13.0)
- Mongo

### **Các bước chạy code:*Mở project và chạy lệnh

   ```
   npm install
   npm start
   ```
Mở trình duyệt và nhập link sau để tạo CSDL:

   http://localhost:3000/product/seed

   http://localhost:3000/user/seed

Truy cập link http://localhost:3000/ để xem trang web
